================================================================================
                        KEY DEVELOPMENT CONCEPTS
================================================================================

STATE MANAGEMENT (Provider Pattern):
- Providers hold app-wide state
- Widgets listen to providers for changes
- When data changes, providers notify listeners
- UI automatically rebuilds with new data

NAVIGATION (GoRouter):
- Uses route paths (e.g., /home, /add-plant)
- Maintains navigation stack
- context.push() adds new route
- context.pop() goes back
- context.go() replaces current route

SUPABASE BACKEND:
- PostgreSQL database for persistent storage
- Real-time authentication (login/signup)
- Storage for image uploads
- Row Level Security (RLS) for data privacy

WIDGETS:
- Reusable UI components
- Immutable StatelessWidget or StatefulWidget
- Take parameters (props) for customization
- Can contain multiple widgets

MODELS:
- Dart classes representing data
- toJson() converts to JSON for database
- fromJson() converts database data to class
- Enable type safety and code organization
